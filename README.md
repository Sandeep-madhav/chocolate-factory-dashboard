# 🍫 Chocolate Shipments Dashboard – Power BI

## 📊 Dashboard Overview

This Power BI dashboard delivers a comprehensive view of chocolate shipment performance, designed to support data-driven decision-making in sales and logistics. It transforms raw shipment data into clear, interactive visual insights, enabling users to quickly understand trends, patterns, and key performance indicators.

The dashboard emphasizes usability, interactivity, and business relevance, making it suitable for both operational monitoring and strategic analysis.

---

## 📈 Key Features

### 🔹 Sales Performance Tracking

* Displays total revenue, shipment volume, and key KPIs
* Tracks performance trends over time (monthly/yearly)
* Identifies peak and low-demand periods

### 🔹 Product-Level Insights

* Highlights top-performing chocolate categories
* Compares revenue contribution across products
* Analyzes quantity vs revenue relationships

### 🔹 Regional Analysis

* Visualizes shipment distribution across regions/countries
* Identifies high-performing markets
* Enables comparison of regional sales performance

### 🔹 Interactive Filtering

* Dynamic slicers for date, product category, and region
* Drill-down capabilities for deeper analysis
* Real-time updates across all visuals based on user selection

---

## 📊 Visual Components

* **KPI Cards** – Quick snapshot of key metrics
* **Line Charts** – Time-based sales trends
* **Bar Charts** – Product and regional comparisons
* **Donut/Pie Charts** – Distribution breakdowns
* **Slicers** – Interactive filtering controls
